# Deploy Jekyll Static Site on Ubuntu 24.04

Jekyll allows developers to build fast, static websites without relying on heavy content management systems. This guide introduces the process of publishing a Jekyll site on a Ubuntu 24.04 server for a simple, low-maintenance hosting setup.

For the complete deployment walkthrough with commands and configuration tips:
https://hostperl.com/kb/tutorials/deploy-jekyll-static-site-on-ubuntu-2404
