# How to Install and Use OpenTofu on Ubuntu 24.04

OpenTofu is an open-source infrastructure-as-code tool designed for managing cloud and on-prem resources through declarative configuration. This guide introduces how to install OpenTofu on Ubuntu 24.04 and start working with infrastructure provisioning workflows.
