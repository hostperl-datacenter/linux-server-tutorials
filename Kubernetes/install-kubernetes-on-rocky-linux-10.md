# Install Kubernetes on Rocky Linux 10 with Cilium

Kubernetes enables container orchestration at scale, while Cilium provides advanced networking and security powered by eBPF. This guide introduces how to deploy a Kubernetes cluster on Rocky Linux 10 and integrate Cilium for enhanced networking and policy control.
