# Install and Configure Uptime Kuma on Ubuntu 24.04

Uptime Kuma is a simple, self-hosted monitoring tool that keeps an eye on websites, APIs, and network services. This guide introduces how to install and configure Uptime Kuma on Ubuntu 24.04 for straightforward uptime tracking and alerts.
